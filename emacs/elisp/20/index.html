Configuration for emacs 20 or up

This emacs configuration works on both Windows NT emacs and Linux emacs environment.

Installation

In linux, the emacs is installed by default. For Windows, please download it first and install. See Here

For Windows NT, follow the following steps:

  • set HOME environment variable.
  • copy the elisp.zip to your HOME dir, and do an 'unzip'.
  • Modify the '_emacs' file in your HOME dir, change the '*HOME*' to your home dir in '_emacs' file.
  • Then start 'emacs', you are all set.

Here are some features:

VI mode

I am addict to vi. I use viper mode (vi keybinding for emacs) inside emacs. Several file format such as .java, .c, .h, .html will automatically use viper mode. To switch between vi mode and emacs mode, use Ctrl-Z.

Java mode

By default, java mode is supported. It supports syntax highlight, smart indent, auto-completion. And if configured well, press [F1], and type in the jdk class you want to load, it will show the class you want. Please change the jdk src root in sun-java.el.

Ctrl-mouse-right-button will show methods in the current class.

Other modes

This configuration also supports html mode, css mode, perl mode, C/C++ mode.

Keybinding

  • Ctrl-left-mouse-button: show files in buffers.
  • Ctrl-right-mouse-button: show methods in a class.
  • F1: Load jdk class
  • F4: run 'grep'
  • F5: show speedbar
  • F6: run 'ff edit', need install Firefly first, and modify some variables.
  • F7: run 'ff delget'.
